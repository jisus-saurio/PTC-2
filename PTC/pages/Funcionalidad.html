¿Como

Funcionamos?

Somos un equipo de programadores, que implementamos nuetros conocimientos para desarrollar un sistema minimalista, para facilitar la administración de un salón de belleza, contribuyendo así a un mejor manejo de la administración.

Tener el conocimiento básico del funcionamiento de una computadora y saber utilizar componentes externos.

Los componentes que el computador necsita para ejecutar nuestro sistema son:
Un procesador RYZEN 5
Mínimo 8GB de RAM
Un disco duro de SCD de 256GB
Sistema operativo deWindows 10x64

Los componentes que recomendamos para ejecutar nuestro sistema son:
Un procesador RYZEN 7
Mínimo 16GB de RAM
Un disco duro de SCD de 500GB
Sistema operativo deWindows 10x64
RTX 3060
